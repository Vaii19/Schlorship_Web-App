# Scholarship Web App

## 📌 Description
This is a web-based scholarship management system that allows students to apply for scholarships and administrators to manage applications.

## 🛠 Technologies Used
- PHP
- MySQL
- HTML/CSS
- XAMPP

## 🚀 Features
- Student application form
- Admin dashboard
- View and manage applications
- Generate award letters

## ▶️ How to Run
1. Download the project
2. Move to `htdocs` (XAMPP)
3. Import database from `sql` folder
4. Start Apache & MySQL
5. Open `localhost/project-name`
